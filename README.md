# Interview Prep Tracker

🎯 **45-Day structured interview preparation tracker** for service-based companies (Infosys, IBM, Accenture, TCS).

## Features
- 📊 Dashboard with streak tracker, progress rings & activity heatmap
- 🗺️ Complete 45-day day-by-day study roadmap
- ⚡ DSA topic checklist (Striver A2Z aligned — 34 topics with LeetCode/GFG links)
- ⚛️ React & JavaScript tracker (24 topics with hooks, closures, Virtual DOM)
- 🧩 OOP Concepts tracker (4 pillars, SOLID, design patterns)
- 📚 10 curated FREE resources (Striver, NeetCode, Namaste JS, react.dev)
- 🎤 Mock Interview Q&A (18 HR + Technical questions with answers)
- 💾 Progress auto-saves to localStorage — no login needed

## How to Use
Just open `index.html` in your browser — or visit the live site!
